# 03 — Trait Mapping

## Overview

This section covers genome-wide association studies (GWAS) and genotype–phenotype mapping using variants derived from the pangenome. Phenotype data used in these analyses are not stored in this public repository.

> **Note:** Sensitive phenotype files are excluded from this repository. Only summary statistics and association results are shared.
